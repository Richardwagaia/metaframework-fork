# Meta Framework Cms
Sub CMS Framework pour Laravel 
### Installation

```bash
composer require aboleon/metaframework`

php artisan vendor:publish --tag=metaframework
```
### Utilisation

#### Components
    
```blade
<x-metaframework::input name="phone" :value="$data->phone" />
```
